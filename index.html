📊 Google Sheet লাইভ সিঙ্ক (Apps Script)

নিচের ধাপ অনুযায়ী আপনার Google Sheet-এ একটা Apps Script বসিয়ে "Web App" হিসেবে ডিপ্লয় করুন, তারপর যে লিংক পাবেন সেটা এখানে বসান। এরপর থেকে এই টুলে যা কিছু আপডেট হবে, তা সরাসরি আপনার Sheet-এ চলে যাবে — কোনো সাইন-ইন লাগবে না।

নিচের ধাপ অনুসরণ করে এই লিংকটা বের করুন।
সংযুক্ত নয়
ধাপে ধাপে — Apps Script কীভাবে বসাবেন
  1. আপনার Google Sheet খুলুন → মেনু থেকে Extensions → Apps Script ক্লিক করুন (আপনার স্ক্রিনশটে যেটা দেখা যাচ্ছে)।
  2. যে কোড এডিটর খুলবে, ওখানে যা লেখা আছে সব মুছে নিচের কোডটা পেস্ট করুন (নিচে দেওয়া আছে)।
  3. উপরে সেভ (💾 disk আইকন) চাপুন, প্রজেক্টের একটা নাম দিন (যেমন: MIM Tracker Sync)।
  4. উপরে ডানদিকে নীল Deploy → New deployment বাটনে ক্লিক করুন।
  5. গিয়ার/সেটিংস আইকনে ক্লিক করে Type থেকে Web app বেছে নিন।
  6. "Execute as" → Me, "Who has access" → Anyone সিলেক্ট করুন। তারপর Deploy চাপুন।
  7. প্রথমবার একটা permission চাইবে — নিজের অ্যাকাউন্ট বেছে "Advanced → Go to (project) → Allow" করুন।
  8. যে Web app URL পাবেন (অনেকটা এমন: https://script.google.com/macros/s/.../exec), সেটা কপি করে উপরের বক্সে বসিয়ে "লিংক সংরক্ষণ করুন" চাপুন।

💡 প্রতিবার কোড এডিট করলে আবার নতুন করে Deploy করতে হবে না — Deploy বাটনের পাশে ✏️ (Manage deployments) থেকে "Edit → New version" করে আগের একই লিংক রেখেই আপডেট করা যায়।

Apps Script কোড (কপি করুন) 
function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Backup') || SpreadsheetApp.getActiveSpreadsheet().insertSheet('Backup');
  var data = sheet.getRange(1,1).getValue();
  return ContentService.createTextOutput(data || '{}').setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  var lock = LockService.getScriptLock();
  lock.waitLock(20000);
  var backupSheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Backup') || SpreadsheetApp.getActiveSpreadsheet().insertSheet('Backup');
  try {
    var raw = e.postData.contents;
    backupSheet.getRange(1,1).setValue(raw);
    var data = JSON.parse(raw);
    writeReadableSheet(data.records || []);
    backupSheet.getRange(1,2).setValue('OK - ' + new Date());
  } catch (err) {
    backupSheet.getRange(1,2).setValue('ERROR: ' + err.message);
  } finally {
    lock.releaseLock();
  }
  return ContentService.createTextOutput(JSON.stringify({status:'ok'})).setMimeType(ContentService.MimeType.JSON);
}

function writeReadableSheet(records) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName('All Records') || ss.insertSheet('All Records');
  sheet.clear();
  var header = ['OFFICE','SL','NAME','PASSPORT','DESTINATION','WAKALA','STATUS','AGENT','NOTE','RETURN STATUS','RETURN DATE','FINE','UPDATED AT'];
  sheet.appendRow(header);
  var counters = {};
  records.forEach(function(r){
    counters[r.office] = (counters[r.office] || 0) + 1;
    sheet.appendRow([r.office, counters[r.office], r.name, r.passport, r.destination, r.wakala, r.status, r.agent, r.note, r.returnStatus, r.returnDate, r.fine, r.updatedAt]);
  });
}

✅ সিঙ্ক ঠিকমতো হচ্ছে কিনা যাচাই করতে: Sheet-এ "Backup" ট্যাবে যান, B1 সেল দেখুন — "OK - তারিখ" লেখা থাকলে সিঙ্ক সফল, "ERROR: ..." লেখা থাকলে সেই বার্তাটা স্ক্রিনশট দিন।
